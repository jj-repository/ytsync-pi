# ytsync-pi

Syncs YouTube playlists/likes to a NAS as MP3 (MusicBrainz-tagged) or MKV video. Rust one-shot binary driven by a systemd timer on a Raspberry Pi. Stack: Rust 2021, rusqlite, yt-dlp + ffmpeg + fpcalc subprocesses.
